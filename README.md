## Overview
This course provides a complete lab environment for learning and practicing:
- **Security Testing** Using OWASP ZAP and WebGoat
- **Performance Testing** using k6
- **Real-world vulnerability exploitation** in a safe environment
- **Load and stress testing** techniques

All tools run in isolated Docker containers, making setup simple and consistent across different operating systems.
